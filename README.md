# Retail Analytics Dashboard

## Overview
An interactive Power BI dashboard developed to analyze retail sales, profit, and customer performance.

## Dashboard Pages
- Sales Analysis
- Profit Analysis
- Demographics Analysis

## Features
- KPI Cards
- Interactive Slicers
- DAX Measures
- Data Modeling
- Power Query
- Navigation Buttons

## Tools Used
- Power BI
- DAX
- Power Query
- Microsoft Excel
